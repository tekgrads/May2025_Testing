# 📘 Day 1: Java Computing Basics

## 💻 What is Computing?
> Any kind of **processing** done by a system.

---

## ☕ Why Java?

- ✅ **Platform Independent**
- ✅ Source code➡️ compiled once ➡️ run **anywhere**
- ✅ Uses **bytecode** as intermediate form
- ✅ JVM interprets bytecode into machine code

---

## 🔄 Java Compilation & Execution Flow


### ✅ Bytecode
- Generated by the Java Compiler (`javac`)
- Stored in `.class` files
- Can be executed on **any platform** with a JVM

### ✅ JVM (Java Virtual Machine)
- Converts **bytecode** to **machine-understandable code**
- Ensures Java runs on **any OS and processor**

### ✅ Platform = OS + Processor

---

## 🧰 Java Components

### 🔹 `javac` – Java Compiler
- Compiles `.java` source code to **bytecode**
- **Platform independent**

### 🔹 `java` – Java Interpreter
- Executes the **bytecode** using the **JVM**
- **Platform-specific** execution

---

## 🧑‍💻 JDK vs JRE

| Component | Full Form               | Purpose            | Used By           | Includes                  |
|----------|--------------------------|---------------------|--------------------|---------------------------|
| **JDK**  | Java Development Kit     | For **development** | Programmers        | JRE + Development Tools   |
| **JRE**  | Java Runtime Environment | For **execution**   | End Users / Clients| JVM + Libraries           |
